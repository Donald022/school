IT Infrastructure Management for Property Management

Project Description

As part of my role with Jake Keo Real Estate Property Management, I developed and implemented comprehensive IT infrastructure solutions for multiple Airbnb rental properties. This project involved setting up secure networks, managing smart devices, and protecting both guests and property owners from cyber threats.

The project required evaluating security vulnerabilities, implementing network security protocols, and establishing standardized device installation procedures across numerous properties. I also created training materials to help property owners understand phishing threats and maintain secure operations.

Key Features

  • Network security implementation for guest Wi-Fi and property systems
  • Smart device installation and configuration (smart locks, thermostats, cameras)
  • Phishing awareness and prevention protocols
  • Standardized IT setup procedures for new properties
  • Remote device management and troubleshooting systems

Skills Learned

Through completing this project, I developed and strengthened the following skills:

  • Network Security: Implemented secure network configurations to protect sensitive data and guest privacy
  • Device Management: Gained expertise in installing, configuring, and managing IoT devices remotely
  • Cybersecurity Awareness: Developed protocols to identify and prevent phishing attacks and other security threats
  • Documentation: Created comprehensive setup guides and security procedures for property managers
  • Troubleshooting: Built systematic approaches to diagnose and resolve technical issues remotely
  • Client Communication: Explained complex technical concepts to non-technical property owners and guests
